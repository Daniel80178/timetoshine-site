The Approach

Why this isn't another auto-bot

The difference between a strategy and a script is judgment. Here's how that judgment is applied to every trade.

01

Decisions before code

The team reads the market every day. Macro context, daily structure, expected news. Only when conditions match the strategy's edge is an entry placed. Patience is a feature, not a bug.

02

Software for what software does best

Position management, grid placement, anchor TP math, news-blackout enforcement — all handled by the bot, in real time, without emotion. Humans set direction; software handles precision.

03

Transparency by default

The live MyFXBook track record is public. The cTrader Copy stats are public. Followers see exactly what's happening, when it's happening — including the losing trades.

04

Profit discipline

Profits are withdrawn at meaningful milestones rather than left to compound indefinitely. This caps the absolute risk on the table at any point in time — and protects the people copying.
